# Stock Agent Dashboard (Web UI)

Professional React 19 + TypeScript dashboard for the Stock Agent System.

## Quick Start

\`\`\`bash
cd web-dashboard
pnpm install
pnpm dev
\`\`\`

## Features

- 7 Pages: Dashboard, Analysis, Portfolio, Backtest, Training, Settings
- TradingView Charts
- Real-time WebSocket updates
- PDF/CSV Export
- OpenBB Natural Language Query
- Voice Input Support
